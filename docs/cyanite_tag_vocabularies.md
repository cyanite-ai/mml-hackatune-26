model outputs (moods, genres, instruments, character, etc.). Use these exact keys
when building tag filters or interpreting model output. Free-text outputs
(free genre, augmented keywords) have no fixed vocabulary and are not listed here.

Source: Cyanite public API documentation (model outputs).
